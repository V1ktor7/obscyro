---
title: FHIR
description: How Obscyro fits into FHIR-based healthcare data flows.
---

Obscyro is FHIR-aware: any concept it returns can be embedded directly into FHIR resources (`Coding`, `CodeableConcept`) without further transformation. The system URI for SNOMED is `http://snomed.info/sct`; for ICD-10 it is `http://hl7.org/fhir/sid/icd-10`.
